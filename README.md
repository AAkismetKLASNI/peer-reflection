# :star:Reflection / VideoChat / Next / WebRTC

Многопользовательский видеочат (до 4 человек в одной комнате), где можно общаться со своими друзьями!

### Основной стек:
- Next
- Bun / Express
- Socket IO
- WebRTC Api
- Typescript
- Jotai
- Uuid
- React-toastify

- bun i (/client)
- bun dev (/client)
- bun i (/server)
- bun start (/server)
